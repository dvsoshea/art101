Task 1: Psuedocoding an everyday task

  • Collect everything dirty from your room
  • Separate light from dark clothing
  • Go to the washer
  • Dump in the first load of laundry
  • Measure out a lid full of detergent
  • Add detergent to washers
  • Close lid
  • Select the cycle on the washer according to your load
  • Press start

Task 2:Psuedocoding a computer task

  1. Print the board before each turn
    1. Retrieve previous board layout
    2. Print board layout
  2. Accept user input
    1. Detect which box the user clicks
    2. Store that information
  3. Validate user move
    1. Check whether the user move is possible
    2. Print user move to board
  4. Evaluate possible moves
    1. Check available spaces
    2. Determine ideal next move
      1. Define an ideal move
  5. Choose computer move
    1. Print move to board
  6. Test for Win! Condition
    1. If true print Win
    2. If false return to step 1
